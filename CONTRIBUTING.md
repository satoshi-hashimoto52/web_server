# Contributing

このリポジトリへの貢献ルールです。軽微な修正でも歓迎します。

## 開発環境
- macOS想定（M1/M2含む）
- Python 3.x
- Node.js 18+ 推奨

## ブランチ運用
- 1つの変更は1ブランチで作業
- 変更内容が分かるブランチ名を推奨（例: `feature/region-results`）

## コーディングガイド
- 既存の構成・命名に合わせる
- 変更範囲は最小にする
- UI文言は日本語を基本とする（タイトル/ロゴ周辺は除く）

## 動作確認
- バックエンド/フロントの起動確認
- 推論領域の作成/移動/リサイズ/削除
- 領域ごとの推論結果が更新されること

## 追加した機能の記録
- 仕様変更は `docs/spec.md` に追記
- API変更は `docs/api.md` に追記
- UI/UXの方針は `docs/ai.md` に追記
- プログラム・コードを更新したら、関連するドキュメントも同時に更新する
- 仕様が不明な場合は、先に確認してから実装する

## 連絡事項
不明点はIssueまたはPRに記載してください。
