# 仕様

## 概要
- Webカメラ/RTSP/HTTPストリームをYOLOv8で推論し、ブラウザにリアルタイム配信する。
- 推論領域はユーザーが画面上で作成・移動・リサイズできる。
- 領域ごとの推論結果は右側リストに表示する。

## 画面仕様
- タイトル/ロゴ周辺以外の操作説明は日本語表記。
- ストリーミング画面は左に映像、右に結果リストの2カラム構成。
- 推論領域は赤枠矩形で表示し、ドラッグで移動、右下ハンドルでサイズ変更。

## 推論領域
- 領域名は重複不可。
- 追加/削除はUIボタンで行う。
- フロント内部はピクセル座標で管理し、WebSocket送信時に百分率へ変換する。
- 領域情報はフロントからWebSocketで随時更新する。

## 推論結果
- 領域内に入った検出ボックスを左端座標の昇順で並べ、ラベルを連結して表示。
- 例: "0","1" → `01`

## 入力ソース
- HTTP/RTSP URL
- Macbookカメラ（`device:0` など）
